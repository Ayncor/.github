# Ayncor `.github`

This repository is the **organization meta home** for [Ayncor](https://github.com/Ayncor) on GitHub.

## What lives here

- **[profile/README.md](./profile/README.md)** — Markdown shown on the **public organization profile** (the special `.github` repository convention).
- Future additions: `CONTRIBUTING.md`, `SECURITY.md`, issue/PR templates under `.github/` if you want repo-wide defaults (optional).

## Local clone

Use any folder name locally; on GitHub the repository name must stay **`.github`** under the **Ayncor** organization.

```bash
git clone https://github.com/Ayncor/.github.git
```
