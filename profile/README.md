## Welcome to Ayncor

**Async-first team communication** — structured around channels, threads and a review-style inbox, with clear identity and org boundaries and **contracts-first** APIs.

### What we’re building

- **Identity & orgs** — signup, membership, roles, invites, audit-friendly setups
- **Core collaboration** — channels, threads, messages, reactions, participants, per-user workflow state
- **Realtime hints** — WebSocket fan-out that nudges clients to refresh; **HTTP remains authoritative**

### Repositories

Explore the org for service repos, client apps, contracts and automation. API shapes and protocols are defined in **contracts**; backend services implement them.

### Contributing

Guidelines and security policies will live in this org repo as we formalize them — PRs welcome once published.
